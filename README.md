# Brick & Razor — лендинг мужского барбершопа

Одностраничный лендинг с записью через Telegram-бот. Часть фриланс-портфолио.

**Стек:** Vite, Tailwind CSS v4, Vanilla JS, Telegram Bot API.

## Запуск локально

```bash
npm install
cp .env.example .env   # подставить токен бота и chat_id
npm run dev
```

## Деплой

Vercel: подключить репо, прокинуть `VITE_TG_BOT_TOKEN` и `VITE_TG_CHAT_ID` в Environment Variables.
